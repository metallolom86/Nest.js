## Installation

```bash
git clone https://github.com/metallolom86/Nest.js.git
cd Nest.js
yarn
```

Setup MongoDB database.

Create file .env, add two variables to it

```
//  /server/.env
JWT_SECRET=
DATABASE=

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
